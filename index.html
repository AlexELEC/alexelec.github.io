    AlexELEC Project #

    This is a set of solutions aimed at convenient setup and comfortable viewing of IPTV channels in the Kodi media center. This set includes:

    • TVLINK IPTV Server - an IPTV channel repeater and aggregator with a user-friendly web interface.

    • TVLINK PVR Client - an add-on for Kodi that allows watching IPTV channels.

    • TVLINK Simple Control - an add-on for Kodi that allows controlling some aspects of TVLINK’s operation.

    • LibreELEC / CoreELEC (ae-fork) - systems that are modified to work with TVLINK.

    • Kodi for Android - a modified version of Kodi (Android packages) with TVLINK support.
