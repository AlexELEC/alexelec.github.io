    Quick Start #

    This implies that all settings are set by default.

    After installing and running the program, enter the address «http://ip-address:2020» in your browser. Where «ip-address» is the address of the computer on which «TVLINK» is installed.

    Activate «Playlists sources» and click the «Add playlist» button.

    In the «Add new playlist» form, specify the playlist name (Playlist name) and the URL address or file path (Playlist path) of this playlist.

    • the playlist name must be in Latin letters, without special characters, and must not start with a digit
    • the file path must start with «file://», for example: file:///Data/Media/playlist.m3u

    After confirmation (Save changes), activate the «Add channels» switch for this channel source and click the «Update» icon.

    As a result of these actions, all channels from your playlist will be added to «TVLINK», specifically to the «Channels» page (so-called «mapping» of channels). This will already be sufficient to start watching channels.

    On the «About» page, you can download the playlist (M3U Playlist) and open it in any IPTV player, for example VLC. Or, on the «Channels» page, in the «ID» column, you can click the blue triangle (Play) and watch the channel directly in the web interface.
