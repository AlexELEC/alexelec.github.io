    Проект AlexELEC #

    Це набір рішень, спрямованих на зручне налаштування та комфортний перегляд IPTV-каналів у медіацентрі Kodi. До цього набору входять:

    • TVLINK IPTV Server - ретранслятор та агрегатор IPTV каналів із зручним веб-інтерфейсом.

    • TVLINK PVR Client - доповнення для Kodi, яке дозволяє дивитися IPTV-канали.

    • TVLINK Simple Control - доповнення для Kodi, що дозволяє контролювати деякі аспекти роботи TVLINK.

    • LibreELEC / CoreELEC (ae-fork) - системи, які модифіковані для роботи з TVLINK.

    • Kodi для Android - модифікована версія Kodi (Android-пакети) з підтримкою TVLINK.
